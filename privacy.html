Privacy Policy

Extension Name: Robots & Indexing Detector

Last Updated: January 31, 2026

Overview

Robots & Indexing Detector is a Chrome extension that helps users analyze publicly available SEO signals such as robots.txt availability, meta robots directives, canonical URLs, and indexing-related indicators for the currently opened webpage.

Information Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data.

  • No personally identifiable information is collected
  • No browsing history is stored or tracked
  • No cookies or analytics are used
  • No user accounts or authentication are required

How Data Is Used

The extension only reads information from:

  • The currently active webpage
  • The publicly accessible robots.txt file of the website

This information is processed locally in the user's browser and is used only to display SEO-related insights. No data is sent to any external server.

Third-Party Sharing

We do not sell, trade, or share any user data with third parties.

Remote Code

This extension does not use remote code. All scripts are bundled directly within the extension package.

Permissions

The extension requests minimal permissions required for its functionality:

  • Tabs: To identify the currently active webpage
  • Host permissions: To access publicly available webpage content and robots.txt files

Policy Updates

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.

Contact

If you have any questions regarding this privacy policy or the extension, please contact the developer through the Chrome Web Store listing.
